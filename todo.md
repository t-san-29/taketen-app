# TODO / アイデアリスト

現在のコード（index.html / style.css / app.js）をベースに、今後追加・改善できる項目をまとめたもの。
優先度は高 / 中 / 低 の3段階。

---

## 機能追加

### 高優先度

- [ ] **結果をタップでコピー**
  スマホで使うとき、変換結果の数値をタップするとクリップボードにコピーされると便利。
  `navigator.clipboard.writeText()` で実装可能。

- [ ] **PWA化（オフライン対応）**
  `manifest.json` + `service-worker.js` を追加するとホーム画面に追加できる。
  現場でWi-Fiのない状況でも使えるようになる。
  静的ファイルのみのため実装コストが低い。

- [ ] **クイック入力ボタン（プリセット）**
  「3寸5分」「1間」「半間(910)」などよく使う値をボタン一発で入力欄にセットできる機能。
  長さタブに特に有効。

### 中優先度

- [ ] **URLパラメータ共有**
  `?cat=length&val=3&unit=shaku` のようなURLで、特定の変換結果を直接開いたり共有できる。
  「この換算結果をLINEで送る」用途に使える。

- [ ] **インチ・フィート対応（英米法）**
  輸入建材・外資系メーカーの資料ではインチ表記が出てくることがある。
  長さタブに `in`（インチ）・`ft`（フィート）を追加。

- [ ] **建材サイズ早見表（別セクション）**
  サブロク板（3×6 = 910×1820mm）、サンパチ板（3×8 = 910×2440mm）など、
  よく使う建材の規格サイズと尺寸表記の対応表。

- [ ] **変換履歴**
  直近5〜10件の変換結果を `localStorage` に保存して表示。
  「さっきの数字なんだっけ」をなくせる。

### 低優先度

- [ ] **重さタブの追加**
  匁（もんめ）・貫 ↔ g・kg。当初スコープ外にしたが、将来的に追加できる。
  `CATEGORIES` オブジェクトにデータを追加するだけで対応可能な構造になっている。

- [ ] **複合単位の入力・表示**
  「3尺5寸」「2間3尺」のように複数単位の合計を入力できるモード。
  現場でのやり取りに近い入力形式。

- [ ] **坪・帖の地域差対応**
  畳のサイズは江戸間・京間・中京間で異なる。
  現在は1畳 = 0.5坪（江戸間基準）に固定されているため、選択できると正確。

## UI改善

### 高優先度

- [ ] **換算表の行・列ハイライト**
  表のセルをホバー/タップしたとき、その行と列を同時にハイライトする。
  大きな表でどの行・列かを追いやすくなる。

- [ ] **モバイルのキーボードを数値専用に**
  現在の `<input type="number">` に `inputmode="decimal"` を追加することで、
  iOSのソフトキーボードがテンキー表示になり操作しやすくなる。
  1行の変更で対応可能。

- [ ] **印刷ボタンの追加**
  CSSに印刷用スタイル（`@media print`）は実装済みだが、ボタンがない。
  「この表を印刷」ボタンを換算早見表カードに追加するだけで完成する。

### 中優先度

- [ ] **ダークモード対応**
  CSSがカスタムプロパティ（`--primary`, `--bg` など）で書かれているため、
  `@media (prefers-color-scheme: dark)` にダーク用変数を追加すれば対応できる。

- [ ] **タブのスワイプ切り替え（モバイル）**
  スマホで左右スワイプするとタブが切り替わるようにする。
  `touchstart` / `touchend` のイベントで実装可能。

- [ ] **変換結果のアニメーション**
  数値が切り替わるときにフェードインやカウントアップアニメーションをつけると
  視覚的なフィードバックが伝わりやすくなる。

- [ ] **フォントサイズ拡大ボタン**
  現場でスマホを見る際、老眼等で文字が見づらいケースがある。
  ヘッダーにフォントサイズ切り替えボタン（標準 / 大）を追加。

### 低優先度

- [ ] **换算表の表示単位フィルタ**
  列数が多い長さの表（9列）を絞り込めるチェックボックス。
  例：「mm / cm / m だけ表示」など。

- [ ] **result-hint の活用**
  CSSに `.result-hint` クラスが定義されているが現在未使用。
  変換結果の下に補足（例：「3寸5分 = 105mm（一般的な柱サイズ）」）を表示できる。
