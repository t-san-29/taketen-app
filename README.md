# 尺メートル法 変換ツール

建築業界で使われる**尺貫法とメートル法**を素早く変換・参照できるWebツールです。
初心者職人や建築に携わる方が、単位の感覚を身につけるための早見表・計算機です。

🔗 **公開URL**: https://t-san-29.github.io/taketen-app/

---

## 概要

「1尺って何センチ？」「10坪は何平米？」
建築現場では尺貫法とメートル法が混在して使われており、職人には当たり前の感覚でも初心者にはとっつきにくいものです。

このツールは **変換計算機と換算早見表** を一画面で提供することで、その曖昧さを解消することを目指しています。

---

## 機能

### 変換計算機
数値を入力して単位を選ぶと、すべての対応単位へ即時変換します。

### 換算早見表
「1尺 = ○○mm = ○○cm = ○○間 ...」という全組み合わせの一覧表。

### 建築ヒント欄
現場でよく使う基準値（1間 = 1820mm、3尺 = 910mm など）をまとめています。

---

## 対応単位

| カテゴリ | 尺貫法 | メートル法 |
|----------|--------|------------|
| 長さ | 厘・分・寸・尺・間・丈 | mm・cm・m |
| 面積 | 畳・坪 | m² |
| 体積・容積 | 合・升・斗・石 | mL・L・m³ |

---

## 換算の基準値

本ツールは計量法に基づく定義値を使用しています。

| 単位 | メートル法換算 |
|------|----------------|
| 1尺  | 10/33 m ≈ 303.03 mm |
| 1寸  | 1/33 m ≈ 30.30 mm  |
| 1間  | 60/33 m ≈ 1818.18 mm |
| 1坪  | (60/33)² m² ≈ 3.306 m² |
| 1升  | 1.8039 L |

---

## 使い方

1. **タブ**（長さ／面積／体積）でカテゴリを選ぶ
2. 数値を入力し、変換元の単位をプルダウンで選ぶ
3. すべての単位への換算結果が即時表示される
4. 下の早見表で全組み合わせを一覧確認できる
