# taketen-app

## プロジェクト概要

建築業界で使われる「尺メートル法」（尺貫法 ↔ メートル法）の変換ツール・一覧表。
初心者職人や建築に携わる人が、長さ・面積・体積・重さの単位をすぐに確認・変換できる静的Webアプリ。

## ターゲットユーザー

- 建築業界の初心者（尺・寸・坪などの感覚がまだフワフワしている人）
- 職人や施工管理者（素早く換算したい人）

## 機能要件

### 対応単位

**長さ**
- 尺、寸、分、厘、間、丈 ↔ メートル(m)、センチ(cm)、ミリ(mm)

**面積**
- 坪、畳 ↔ 平方メートル(m²)

**体積・容積**
- 合、升、斗、石 ↔ リットル(L)、立方メートル(m³)


### 主な機能

1. **変換計算機** - 数値を入力して即時変換
2. **一覧表** - 単位系ごとの換算表（見やすく・印刷対応）
3. **早見表** - よく使う基準値（1尺=○○cm など）

## 技術スタック

- **HTML / CSS / JavaScript**（バニラJS、フレームワーク不使用）
- ビルドツール不要・依存パッケージなし
- GitHub Pages でそのまま公開可能
